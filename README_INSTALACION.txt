PROVICHECK Enterprise v0.90

Archivos incluidos:
- app.py
- pages/01_Equipos.py
- pages/04_Verificaciones.py
- utils/ui.py

Instalación:
1. Haga una copia de seguridad del repositorio.
2. Reemplace los cuatro archivos conservando sus rutas.
3. Ejecute la aplicación y pruebe Dashboard, Equipos y Verificaciones.
4. Si todo funciona, haga commit y push a GitHub.

Cambios:
- Tema corporativo verde Providencia + azul Incauca.
- Contraste global corregido.
- Sidebar corporativo unificado.
- Encabezado degradado.
- Métricas, formularios, botones y tablas más legibles.
- Equipos PRO mejorado.
- Cada punto de verificación queda dentro de una tarjeta.
- Dos tarjetas de verificación por fila.

Importante:
Esta entrega no cambia la lógica SQLite ni las consultas existentes.
La persistencia externa de documentos se implementa por separado porque requiere credenciales y autorización de Google Drive, OneDrive o SharePoint.
