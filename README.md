# PROVICHECK Enterprise Beta 0.1

Sistema Inteligente de Gestión de Equipos de Laboratorio.

## Incluye
- Login limpio sin mostrar credenciales.
- Dashboard ejecutivo inicial.
- Lectura del Excel `PROVICHECK_Base_Datos.xlsx`.
- Módulo visual de Equipos.
- Hoja de Vida Digital inicial.
- Tendencia demostrativa con límites superior, inferior y valor patrón.
- Administración para revisar hojas del Excel.

## Ejecutar
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Lema
La confianza en un laboratorio comienza con la confianza en sus equipos.
